STEP 1

1. 关于整型常量的默认类型

在JAVA中,整型常量默认的类型是 int ,理论上说 byte b = 100 这种表达式是有问题的,但是JAVA对这种情况进行了特殊处理,
如果数据大小没有超过 byte/short/char 的表示范围,则自动进行类型转换.

同样由于默认的整型常量是int类型的,因此声明 long 变量注意一点,
long a = 10000000000(此数据大于 int 的最大值)
会导致报错,正确的方式是在10000000000后边加上一个L,则不会引发报错
long a = 10000000000L;

2. 关于浮点类型

浮点数常量默认的类型为 double 类型
float a = 1.23;
将会产生报错,因此需要:
float a = 1.23f;

支持科学计数法 12e3表示12乘以10的3次方 12000

值得思考的问题:
float 类型占4个字节, long 类型占8个字节,为什么 float 表示的范围要比 long 大?

7. 垃圾回收机制

调用gc()函数不能直接调用垃圾回收的过程, 只是提供一个请求,该请求是否能够达成,取决于垃圾回收机制.

8. JAVA的构造方法/构造器

构造方法用于构造类的实例,构造方法地位特殊:

  • 构造方法通过 new 关键字调用
  • 构造方法不需要也不能有返回值(因此也不需要 return 语句)
  • 构造方法的名称必须和类名完全一致
  • 如果没有构造方法,编译器会自动添加一个无参的,空的构造方法,如果添加了构造方法,编译器就不会自动添加任何构造方法.

构造方法一般用来初始化对象的各个属性
